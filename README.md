# Park Dong Jin - Portfolio

Web Full-Stack Developer 박동진의 개인 포트폴리오 웹사이트입니다.

## Tech Stack

- **Framework**: [Astro](https://astro.build/) 5.0
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Template**: [AstroWind](https://github.com/onwidget/astrowind)

## Features

- Single Page Application
- Dark Theme (Lime Green + Black)
- Responsive Design
- SEO Optimized

## Sections

- **Hero**: 소개 및 프로필 사진
- **About Me**: 간단한 자기소개 및 연락처
- **Skills**: Front-End, Back-End, Database, Etc
- **Biography**: 경력 및 학력 타임라인
- **Projects**: 프로젝트 포트폴리오
- **Contact**: 연락처 CTA

## Getting Started

### Prerequisites

- Node.js 18+
- npm

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

### Development

개발 서버 실행 후 `http://localhost:4321` 에서 확인할 수 있습니다.

## Project Structure

```
src/
├── assets/
│   ├── images/       # 이미지 파일
│   └── styles/       # CSS 파일
├── components/
│   ├── ui/           # UI 컴포넌트
│   └── widgets/      # 위젯 컴포넌트
├── layouts/          # 레이아웃
├── pages/            # 페이지
│   └── index.astro   # 메인 포트폴리오 페이지
└── config.yaml       # 사이트 설정
```
